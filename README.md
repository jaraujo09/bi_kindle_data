# Fundamentos de Business Intelligence

Este repositório contém o projeto prático desenvolvido para a Unidade Curricular de Fundamentos de Business Intelligence e Análise de Dados. O objetivo é implementar uma solução completa de BI, desde o tratamento de dados até à visualização estratégica.


### Resumo do Projeto
O trabalho consiste na análise de um dataset organizacional  para apoiar a tomada de decisão através das seguintes etapas:

   - **Extração e Tratamento (ETL)**: Limpeza e normalização de dados utilizando Power Query.


   - **Modelação:** Criação de um modelo dimensional (factos e dimensões) no Power BI.


   - **Analytics:** Exploração de Data Mining e algoritmos em Python, para fazer previsões em machine learning


   - **Visualização:** Dashboards interativos com KPIs e indicadores de desempenho.

### Tecnologias

**Power BI:** Power Query & DAX
**Python: **Analytics e Data Mining
**Dataset**: Obtido de fontes abertas (Kaggle).
